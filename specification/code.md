# Qt

1. 变量

   1. 每行声明一个变量。
   2. 每个变量名为功能＋类别的组合，如：用户名行编辑器 = `usrLineEdit`。
   3. 每个变量都要有意义。
   4. 采用驼峰命名法，如：第一个单词以小写字母开始；从第二个单词开始以后的每个单词的首字母都采用大写字母。
2. 宏
 
   1.全部字母大写。 
3. 二元操作符

   1. 左右都要有空格。
4. 逗号

   1. 左边没有空格，右边有空格。
5. 注释

   1. 左右都有空格。
6. 函数

   1. 关键字(如`if`、`for`)右侧有空格。
   2. 使用紧贴花括号，只有一行的控制语句也要写花括号，如：
                                
            if (){
                accept();
            }
            else{
                refuse();
            }
7. 圆括号
   
   1. 为每一个子逻辑使用圆括号进行分组，如：
            
            ((a > b) && (b > c))
